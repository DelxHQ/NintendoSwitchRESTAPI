# Open Source Dependencies

This is a list of all the respositories listed under the "acknowledgements" link at the bottom of the splatoon pages in the the app, and settings menu from the main page of the app. Since those pages only list the licenses, I have compiled a list with the github/npm links. 

Based on the open source dependencies, I think we can safely assume that the Nintendo is using Node for the back-end API, and React for the front-end web view in the Splatoon section of the app. The main app itself is built using Xamarin (C#). Opus is used as the codec for the voice chat features.
